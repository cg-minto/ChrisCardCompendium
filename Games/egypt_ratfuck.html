Customising Combos

There is a bunch of combos to choose from, Its probably best new players should start with only the first 3 combos in play and add more as you get comfortable with the game. If you want an extra twist, have the winning player of each round choose which combo to add.

Combo Description
Double 2 Cards with the same face value played back to back
Sandwich 2 Cards with the same face value played with any card between them
Ladder 2 Numbered cards played in order (e.g., 5, 6)
Chute Numbered cards played in reverse order (e.g., 8, 7)
Big Mac Top card matches the card at the bottom of the deck (often a trickier one to pull off!)
Flush 3 cards of the same suit played back to back
Lovers King and Queen played back to back
Harlot Queen and Jack played back to back
Brothers Jack and King played back to back
